Mi experiencia

Desde el año 2012 y hasta la fecha me dedico a realizar y mantener sitios web para varios sectores de turismo, entre ellos hoteles, agencias de viajes. También para servicios turísticos como transportes, renta de yates. Estos sitios cuentan con un motor de reservaciones, pasarela de pagos y calendario de disponibilidad. (Actualmente usamos laravel 8, Vue Js, Mysql, Bootstrap CSS).
El año 2020 y este 2021 me contactaron para realizar módulos (codeigniter 3, Jquery, mysql) de una página para una transportista y bodega de contenedores asociada a la aduana (entrada, salida de mercancía). En esta misma empresa elabore un API (laravel 5.6, mysql) para permitir a una empresa de transporte marítimo tener acceso por medio de sus aplicaciones a todos los movimientos que se realizan en la transportista con sus contenedores y otras actividades.
